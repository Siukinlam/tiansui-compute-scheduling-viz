# 算力调度过程可视化场景

这是一个用于客户理解算力调度过程的静态交互场景。页面展示：任务排队、调度判断、GPU 分配、开始运行。

## 运行方式

直接双击或浏览器打开：

```bash
open index.html
```

也可以用本地静态服务器运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 宣讲建议

- 先停留在“未调度”，说明任务等待和 GPU 空闲可能同时发生。
- 点击“播放调度过程”，让客户看任务、调度器和 GPU 状态如何变化。
- 切到“运营中”，说明任务运行后可以被隔离和追踪。

## 内容来源

场景信息来自用户提供的宣讲材料，已简化为调度过程演示。
